Code for Kagoshimaとは？

Code for Kagoshimaとはシビックテック（Civic Tech）1の考えに基づき、Codeの力で地域課題の解決を目指す有志による任意団体です。 主に鹿児島県を活動域とし、アプリの開発やデータの利活用、ICTリテラシーの向上について努めていきたいと考えています。

※既に鹿児島県新型コロナウイルスまとめサイトを運営されている#CodeforKagoshima とは別の団体になります。

理念

ITを活用しながら、自分たちの地域の課題を自分たちで解決していく、コミュニティをめざします。

活動方針

  • 参加者の主体的活動を基本とします。
  • 原則、月１回の開催をめざします。
  • 勉強会や交流会等を通して鹿児島県のオープンデータの利活用を促進します。

Facebookページ でも活動を公開しています。お気軽にお問い合わせください。
