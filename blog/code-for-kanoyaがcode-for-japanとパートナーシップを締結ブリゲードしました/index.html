ご報告が遅れてしまいましたが、12/1にCode for KanoyaがCode for Japanとパートナーシップを締結（ブリゲード）しました😆

これまで活動してきた内容がCode for Japanと同じく、ITで地域課題の解決に挑戦する「Code for X（地域名）」として認められたことを嬉しく思います🤩

各ブリゲードの規模や活動テーマ、活動内容は様々です🤓Code for Kanoyaはオープンデータを収集して鹿屋市はもとより鹿児島県、九州、全国に貢献できるアプリを創出するというポリシーを掲げています😇

以下のページの最下部にブリゲードがリストアップされていますので、興味のある方はご覧ください😀

https://www.code4japan.org/brigade
