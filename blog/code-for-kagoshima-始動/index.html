Code for Kagoshima を発足いたしました！

Code for XX とはITで地域課題の解決に挑戦する団体であり、全国に様々な団体があります。 （Code for XX についての詳細はここでは省略いたします。興味のある方はCode for Japanの公式サイト をご覧ください。）

昨今の新型コロナウイルス禍ではCode for Japan をはじめ各地のCode for XX の活動により、感染者数等を可視化したサイトやテイクアウト支援アプリなど様々な活動が迅速に行われました。

そのような中、鹿児島県におきましても行政に頼るだけでなく、市民が自ら課題解決を目指すCode for XX の必要性を強く感じ、当団体を設立いたしました。

今後、定期的に活動を続け、鹿児島県の地域課題解決に少しでも寄与できるように邁進したいと思います。

どうぞよろしくお願いいたします。
