C4K会議_第3回

2020-08-04 Code for Kanoya

1 分で読めます

昨日、定例ミーティングを開催しました🍈協議事項は以下のとおり。
①新コロ第2波が来ているので、フィジカル空間での周知活動はしばらくできませんね🍐まずはEightに名刺を登録しようということになりました🍊今年中にオンラインセミナーを開催したいなあと考えています。
②コードの力で解決してほしい課題を募集するフォームを作成しました🍋
https://docs.google.com/forms/d/126nNEZcGNhvJKy_7CA9WY9GiiHOjaMtTFlrGMgMWusM/viewform?edit_requested=true
 ③当面の活動プログラムとしては、オープンデータを使ったものにしようと思います🍌九州にはBODIK ODCSというオープンデータカタログサイトがあり、多くの市町村が参画しています🍉ここから地域に貢献できるアプリを生み出せるよう取り組んでいきます。
https://odcs.bodik.jp/
