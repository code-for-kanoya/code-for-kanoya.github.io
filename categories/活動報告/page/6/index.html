C4K会議_第2回

2020-07-20 Code for Kanoya

1 分で読めます

本日、定例ミーティングを行いました🐶前回から1名増えて6名、これから徐々に仲間を増やしていきたいと思います🐱協議事項は以下のとおり。 ①Code for Kagoshimaの名刺が出来ました🐭鹿児島県から西郷どんのキャラクター使用許可もいただき、なかなかの出来栄え🐹これを周知活動に使っていきたいと思います。 ②次の活動プログラムについては、概ね決まりました🐰内容については今後のお楽しみ🦊鹿児島県庁や鹿児島市役所などにも今後ご意見を伺っていくつもりです。 ③私たちが取り組むべき地域課題を発掘するのは、意外と一苦労です🐻普段感じていることでもプロブレムソルバーとして考えると・・・そこで、コードの力で解決してほしい課題を募集するフォームを作成することにしました🐼 今後の活動もお楽しみに🐨

C4K会議_第1回

2020-07-06 Code for Kanoya

1 分で読めます

本日、初めてのミーティングを行いました😆しかし大雨で集まったのは5名のみ😢ディスカッションしたのは以下の事項です。 ①参加メンバーの募集・拡充→名刺を作成して各自周知活動を行うことにしました😀 ②飲食店支援プログラムに続く次の活動プログラム発掘→次回までに一人一案持ち寄ることにしました🤔 ③他ブリゲイド団体から鹿児島に横展開できるプログラムを探索→コロナの第2波が押し寄せているため、その対策を重点目標とします😤 ④県内自治体との協働に向けた個別意見交換・全体イベントの開催→自治体への訪問活動を計画することにしました😌 高校生からもミーティングへの参加申請が来ていて、ちょっと嬉しいです😊今後も隔週月曜日17時半から以下の場所＆Skypeで開催しますので、多くの方との出会いを楽しみにしています😇
